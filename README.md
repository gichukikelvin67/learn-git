## **H2 Description**

**This project was created to practice and understand the basics of Git and GitHub version control. It includes HTML and CSS files used to build a simple web project while learning how to initialize repositories, track changes, commit updates, and push code to GitHub.**

*    Understand GitHub repository management.
*   Build confidence in collaborative development.
